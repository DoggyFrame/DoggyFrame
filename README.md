# DoggyFrame

Version 0.0.1

A front-end workflow and basic framework based on [Grunt](http://gruntjs.com/).

## Quick Start

1. `fork` and `git clone` or download as zip.
2. `unzip` and `cd`.
3. `npm install`.
4. Now you can use grunt tasks. See ui_guide.html for style and js demo.

## Grunt Tasks

- HTMLmin
- HTMLhint
- Jsmerge
- Uglify
- JShint
- Sass
- Sprite
- Imagemin
- Watch
